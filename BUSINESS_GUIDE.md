## 2. 公開（無料ホスティング）

### 推奨：Vercel（最も簡単）

```
1. https://vercel.com でアカウント作成（無料）
2. GitHubアカウントと連携
3. uranaiフォルダをGitHubにアップロード
4. Vercelで「New Project」→ GitHubリポジトリを選択
5. Deploy → 完了（URLが発行される）
```

**メリット**
- 独自ドメイン設定が無料
- `https://` 対応（SEOに必須）
- GitHubにpushするだけで自動更新

### 代替：GitHub Pages

```
1. GitHubリポジトリを作成（例: uranai）
2. Settings → Pages → Branch: main → Save
3. https://ユーザー名.github.io/uranai/ で公開
```

### 独自ドメインの取得（任意・推奨）

| レジストラ | 費用 | 備考 |
|-----------|------|------|
| お名前.com | 年1,000〜2,000円 | 日本最大手 |
| Cloudflare Registrar | 年1,000円前後 | 最安値水準 |

**おすすめドメイン例**
- `uranai-no-yakata.com`
- `mystic-uranai.jp`
- `fortune333.com`

## 3. 収益化の方法

### 方法①：Google AdSense（最も現実的）

広告をサイトに貼り、閲覧数に応じて収益が発生する。

**手順**
1. `https://adsense.google.com` で申請
2. サイトのURLを登録（独自ドメイン推奨）
3. 審査通過後、各HTMLファイルに広告コードを貼る

**収益目安**

| 月間PV | 想定月収 |
|--------|----------|
| 10,000 | 1,000〜3,000円 |
| 100,000 | 10,000〜30,000円 |
| 1,000,000 | 100,000〜300,000円 |

> 占い系はクリック単価が高いジャンル（美容・恋愛広告が表示されやすい）

**広告を貼る推奨箇所**
- 結果表示の直前（最も見られる）
- 結果カードの下
- フッター上部

---

### 方法②：アフィリエイト

占い結果と連動した商品リンクを掲載する。

| ASP | おすすめ案件 |
|-----|------------|
| もしもアフィリエイト | 占い書籍（Amazon） |
| A8.net | 電話占いサービス |
| バリューコマース | パワーストーン・占いグッズ |

**高単価案件例**
- 電話占いサービスへの誘導：**1件あたり2,000〜5,000円**
- 占い師マッチングアプリ：**1件あたり1,000〜3,000円**

**実装方法**
結果ページの下部に「もっと詳しく占ってみたい方へ」として電話占いサービスへのリンクを設置する。

## 4. 集客・マーケティング

### SNS運用（コストゼロ）

#### X（旧Twitter）
- 毎日の占い結果を投稿
- 「今日の星座ランキング」などのコンテンツ
- Xシェアボタンからの自然流入を育てる
- ハッシュタグ例：`#占い` `#タロット` `#星座占い` `#数秘術`

#### TikTok / Instagram Reels
- 占い結果を画面録画して投稿
- 「あなたの運命数は？」系のショート動画
- 占い系はエンゲージメントが高い

#### Pinterest
- 占い結果のスクリーンショットが拡散しやすい
- 画像として保存・シェアされる文化がある

---

### SEO対策

各HTMLの `<head>` に以下を追加することで検索流入が増える。

```html
<!-- seiza.htmlの例 -->
<meta name="description" content="星座と血液型の組み合わせで今週の運勢を無料占い。おひつじ座からうお座まで12星座×A・B・O・AB型の48パターン。">
<meta property="og:title" content="星座×血液型占い | 占いの館">
<meta property="og:description" content="あなたの星座と血液型で今週の運勢を鑑定します">
<meta property="og:image" content="https://あなたのドメイン/ogp.png">
```

**狙うキーワード例**
- 「星座 血液型 占い 無料」
- 「数秘術 運命数 無料」
- 「タロット占い 無料 今日」

## 8. 法的注意事項

### 必須対応

1. **免責事項の明記**
   各ページにすでに「娯楽目的のコンテンツです」と記載済み。これを維持する。

2. **プライバシーポリシーの作成**
   AdSense・Google Analytics を使う場合は必須。以下の内容を記載する専用ページを作る。
   - クッキーの使用について
   - 広告の表示について
   - お問い合わせ先

3. **特定商取引法に基づく表記**
   有料サービスを始める場合のみ必要。無料の間は不要。

### 注意点

- 「当たる」「運命が変わる」など断定的な表現は避ける
- 高額商品の購入を促す表現は景品表示法に抵触する可能性がある
- あくまで「娯楽・エンタメ」として運用する
